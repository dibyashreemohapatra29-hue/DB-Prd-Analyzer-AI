🔍 Missing Edge Cases

5 missing

    Concurrent sessions

    What happens when the same user logs in from two devices simultaneously? Are sessions independent or mutually exclusive?

  • 02
    Token expiry during active use

    If a user is mid-form when their session expires, what is the expected behavior — silent refresh or hard redirect?

  • 03
    Email address change

    Spec does not address what happens to existing sessions if a user changes their email address.

  • 04
    Account deletion mid-flow

    If an account is deleted while the user is logged in, the session must be invalidated — this is unspecified.

  • 05
    Offline / network failure during auth

    Network interruption during OAuth redirect is not covered. User experience for this state is undefined.

✨ Tips for Better PRDs

  • Define explicit acceptance criteria for every feature.
  • Specify error states alongside happy paths.
  • Include concrete numeric values — never "reasonable" or "quick".
  • Name every actor, system, and external dependency.
